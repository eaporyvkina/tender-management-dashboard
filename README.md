## 🎯 Бизнес-задача, решение и результат

**Контекст:** Компания управляет проектами в области теплосетей, водоснабжения, котельных и насосных станций в 4 регионах УрФО.  

**Ключевые боли:**
- Нет единого окна для контроля статусов, бюджетов и сроков.
- Риски выявляются слишком поздно – проблемные проекты обнаруживаются постфактум.
- Перегрузка менеджеров – отсутствует прозрачность распределения нагрузки.
- Отчётность занимает дни – данные разрознены, подготовка ручная.

**Цель:** создать инструмент, который консолидирует разрозненную информацию и позволит принимать эффективные бизнес-решения на основе данных за несколько минут.

**Решение:** Разработать интерактивный дашборд в Power BI, построенный по звёздной схеме, который объединяет ключевые метрики и предоставляет единое пространство для мониторинга текущего состояния портфеля и фокусировки внимания на критических зонах:
- KPI-панель с трендами и сравнениями
- Матрица рисков (бюджет × сроки)
- Bullet Chart для сравнения план/факт/прогноз по проектам
- Загрузка менеджеров и топ-3 проблемных проекта

**Результат:** Дашборд стал центральным инструментом для оперативных и стратегических решений. Руководство получило единое пространство для мониторинга, которое сократило время на анализ с нескольких дней до нескольких минут. Риски по проектам выявляются оперативно, время подготовки отчётов сократилось с 3 дней до 15 минут, доля проблемных проектов снизилась, перегрузка менеджеров устранена.

## 📊 Архитектура данных

### Схема «Звезда»
- **Факт-таблица:** `fact_tenders` (проекты с датами, бюджетами, статусами)
- **Измерения:** `dim_manager`, `dim_status`, `dim_priority`, `dim_project_type`, `dim_region`, `Dates`
- **Вспомогательная таблица:** `Budget_by_Month` (равномерное распределение бюджета по месяцам для корректного тренда)

 Звездная схема обеспечивает быструю фильтрацию, простоту DAX и масштабируемость.

 Данные синтезированы на основе паттернов реального бизнеса, структура приближена к CRM.

## 📈 Ключевые метрики и визуализации

### 1. Executive Summary (главная страница)
- **KPI:** общее количество проектов, бюджет, проблемные проекты, освоение бюджета.
- **Распределение по типам и регионам** – горизонтальные гистограммы.
- **Загрузка менеджеров** – таблица с условными гистограммами (data bars).
- **Топ‑3 проблемных проекта** – список с менеджерами и причинами.

### 2. Финансовый анализ
- **Bullet Chart** – сравнение текущих затрат с планом и прогнозом по каждому проекту.
- **Мера `Monthly Budget`** – сумма распределённого бюджета по месяцам для тренда.

### 3. Матрица рисков (точечная диаграмма)
- **Ось X:** отклонение бюджета (%)
- **Ось Y:** дней до дедлайна
- **Размер точки:** бюджет проекта
- **Цвет:** статус (проблемные выделены красным)
